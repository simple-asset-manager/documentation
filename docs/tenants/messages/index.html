Message Your Property Manager

It is easy for landlords and tenants to make requests, keep in touch, and review previous communications with Messages.

Messages works similar to most modern messaging applications, with each stream of messages contained in its own thread.

To message your property manager:

  1. Select Messages from the navigation panel.
  2. Click "New Message".
  3. Enter the subject for the message thread and click "Create Thread".
  4. Select the newly created message thread from the list to access that message thread.
  5. Type your message in the Message Box and click "Send Message".
